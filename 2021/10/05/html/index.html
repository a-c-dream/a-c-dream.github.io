html

实体

HTML中不能直接写一些特殊的字符，如空格，<字母>等，若要书写可以使用实体（转义字符）

实体语法：

​ &实体名字；

例如：

​

空格 &nbsp;
> &gt;
< &lt;
版权符号 &copy
tab &#9;

meta

设置网页的元数据，对用户无意义，对浏览器和搜索引擎有意义。

常用属性：

  1. charset 设置字符集
  2. name 设置关键字（搜索引擎搜索的依据）多个关键字之间用**,**隔开
  3. content 设置关键字的值

常用功能：

设置网页描述

设置网页重定向

blockquote

表示引用（缩进）

块元素中可以放行内元素，行内元素内不能放块元素，p标签内不能放任何元素

结构

head 网页头部

main 网页主体

footer 网页底部

nav 导航栏

aside 和主体相关的其他内容（侧边栏）

article 一个独立的文章

section 一个独立的区块

div 一个区块

span 行内元素，一般用于在网页上选中文字

列表

ul 无序列表

ol 有序列表

dl 定义列表

  1. dt表示定义内容
  2. dd表示对内容解释说明

超链接

标签：

a

属性：

href

超链接是行内元素，在a标签内可以嵌套除他自身之外的所有标签

target

_self 默认值 在当前页面中打开超链接

_blank 在一个新的页面中打开超链接

例：

href = “#” 跳转到顶部

href = “Javascript:; 点击时什么也不发生

id

元素的唯一标识（每一个标签都可以有）

路径：

./ 当前目录（可以省略）

../ 上一级目录

图片

标签：

img

属性：

src 指定图片路径

alt 如果图片无法显示会用alt值替换（部分浏览器支持）

width 设置图片宽度

height 设置图片高度

注：

一般只设置高度和宽度中的一个（等比例缩放）

不建议缩放

图片的格式：

jpeg(jpg)

-支持的颜色比较丰富，不支持透明效果，不支持动图

-一般用来显示照片

gif

-支持的颜色比较少，支持简单透明，支持动图

-颜色单一的图片，动图

png

-支持的颜色丰富，支持复杂透明，不支持动图

-颜色丰富，复杂透明图片（专为网页而生）

webp

-谷歌推出的专门用来表示网页中的图片的一种格式

-具有其他格式的优点，而且文件还特别小

-缺点：兼容性不好

base64

-将图片使用base64编码，这样可以将图片转化为字符，通过字符的形式引入图片

-一般都是一些需要和网页一起加载的图片才会使用base64

效果一样，用小的

效果不一样，用效果好的

内联框架

向当前页面中引入一个其他的页面

标签：

iframe

属性：

src 引入页面的路径

frameborder 内联框架的边框
